\# 🏠 House Price Predictor



A machine learning web app that predicts California housing prices.



\## 🔗 Live Demo

👉 https://house-price-predictor.vercel.app



\## 🛠 Tech Stack

\- \*\*Frontend:\*\* React, Vite

\- \*\*Backend:\*\* FastAPI, scikit-learn

\- \*\*Model:\*\* Linear Regression (California Housing Dataset)

\- \*\*Deployment:\*\* Vercel (frontend), Render (backend), Docker



\## 🚀 Run Locally

\### Backend

cd backend

docker build -t house-price-api .

docker run -p 8000:8000 house-price-api



\### Frontend

cd frontend

npm install

npm run dev
